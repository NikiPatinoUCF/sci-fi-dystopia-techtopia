# TECHTOPIA ARCHIVES | Dystopian Reading Tracker

AI Humanities Activity

A dystopian/techtopian-themed web application for tracking your sci-fi reading journey through 100 classic and modern books.

## Features

- **100 Curated Books**: Spanning from 1895 to 2017, featuring diverse authors and languages
- **Local Storage Tracking**: All data stored locally in your browser
- **Rating System**: Rate books with a 5-star system
- **Read Status**: Mark books as read or unread
- **Advanced Filtering**: Filter by status, rating, era, and themes
- **Search Functionality**: Search by title, author, description, or themes
- **Multiple Sort Options**: Sort by title, author, year, or rating
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
- **Dystopian Aesthetic**: Cyberpunk-inspired UI with glitch effects and neon accents

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with animations and responsive design
- **Vanilla JavaScript**: No dependencies, optimized performance
- **Local Storage API**: Client-side data persistence
- **Google Fonts**: Orbitron and Share Tech Mono

## How to Use

1. Browse through 100 sci-fi dystopian and techtopian books
2. Click on any book card to view details
3. Rate books using the 5-star system
4. Mark books as read or unread
5. Use filters to find specific books by status, rating, era, or themes
6. Search for books by title, author, or keywords
7. All your reading data is saved automatically in your browser

## Local Development

Simply open `index.html` in your web browser. No build process or dependencies required.

## GitHub Pages Deployment

This site is optimized for GitHub Pages deployment:

1. Push to your GitHub repository
2. Go to Settings > Pages
3. Select the branch to deploy (usually `main`)
4. Your site will be live at `https://[username].github.io/[repo-name]/`

## Data Structure

Each book includes:
- Title and author
- Year of publication
- Original language
- Detailed description
- Thematic tags
- User rating (0-5 stars)
- Read status (read/unread)

## Browser Compatibility

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## Privacy

All reading data is stored locally in your browser using the Local Storage API. No data is transmitted to any server. Clear your data anytime using the "CLEAR ALL DATA" button.
